###编译

在当前目录下使用`make install`编译整个项目;

使用`make run`将会通过VirtualBox启动系统, 你需要自行在系统中安装VirtualBox并创建一个从项目目录的img文件启动的虚拟机,
并将虚拟机的快捷方式替换makefile中的变量SHORTCUT.
